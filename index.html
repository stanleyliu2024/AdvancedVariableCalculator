Welcome To My Variable Storing calculatorBy Stanley Liu

This calculator stores your favorite variables and allows you to use them in the calculator. On the left, you will see a normal calculator, just type in or click the buttons of what you want to be solved. On the right, there is the variable starting part. Type in the variable key and value and then click the verify button to verify that those are valid.

  • All spaces will be ignored so 1 2 will be 12
  • Functions like sqrt and abs must have () after
  • An open parenthesis must have a closing
  • Keys must ONLY contain alphabetical characters
  • Values can be expressions only with operations and numbers like 5 * 3
  • If keys are duplicated, the latest value will be taken
  • Currently, multiple decimals in one number will not be rejected
  • Only key value pairs that were verified and highlighted green are usable
  • G = 6.677 * 10^-11
  • e = 1.6 * 10^-19
  • c = 3.00 * 10^8
  • m = 1.67 * 10^-27
  • epsilonNaught = 8.85 * 10^-12
